# ܡܠܟܬܐ ܕܪܕܝܢܐ ܕܐܪܡܝܐ - Aramaic RTL Guidelines

ܗܢܐ ܗܘ ܡܠܟܬܐ ܕܪܕܝܢܐ ܕܐܪܡܝܐ ܕܡܫܡܚ ܠܟ ܠܡܦܠܚ ܥܡ ܐܪܡܝܐ ܒܡܛܟܣܐ ܕܡܢ ܝܡܝܢܐ ܠܣܡܠܐ (RTL).

## ܐܣܟܪܝܒܘܬܐ ܕܣܘܪܝܝܐ (Syriac Script)

### ܐܣܟܪܝܒܘܬܐ
- ܐܪܡܝܐ ܡܫܬܡܚܬܐ ܒܐܣܟܪܝܒܘܬܐ ܕܣܘܪܝܝܐ (ܐܪܡܝܐ)
- ܐܣܟܪܝܒܘܬܐ ܕܣܘܝܪܐ: ܐ ܒ ܓ ܕ ܗ ܘ ܙ ܚ ܛ ܝ ܟ ܠ ܡ ܢ ܣ ܥ ܦ ܨ ܩ ܪ ܫ ܬ
- ܐܣܟܪܝܒܘܬܐ ܕܐܢܓܠܝܐ ܡܫܬܡܚܬܐ ܓܝܪ ܒܐܣܟܪܝܒܘܬܐ ܕܠܐܛܝܢܝܬܐ ܠܬܟܢܝܬܐ ܕܡܛܟܣܐ

### ܐܣܟܪܝܒܘܬܐ ܕܬܟܢܝܬܐ
- ܐܪܡܝܐ ܗܝ ܠܫܢܐ ܕܡܢ ܝܡܝܢܐ ܠܣܡܠܐ (RTL)
- ܟܬܒܐ ܡܫܪܝ ܡܢ ܝܡܝܢܐ ܘܡܬܡܫܟ ܠܣܡܠܐ
- ܐܢܓܠܝܬܐ ܘܐܝܬܝܗ ܠܫܢܐ ܕܡܢ ܣܡܠܐ ܠܝܡܝܢܐ (LTR) ܒܐܣܟܪܝܒܘܬܐ ܕܠܐܛܝܢܝܬܐ

## ܐܣܟܪܝܒܘܬܐ ܕܒܝܬܘܪܢܛ (BitTorrent Terminology)

### ܐܣܟܪܝܒܘܬܐ ܕܡܬܩܒܠܢ (Transliteration)
- **Torrent**: ܛܘܪܢܛ (ܐܝܬܝܗ ܐܣܟܪܝܒܘܬܐ ܕܡܬܩܒܠܢ)
- **DHT**: ܕܝܛܝܛ (Distributed Hash Table)
- **PEX**: ܦܝܐܟܣ (Peer Exchange)
- **BEP**: ܒܝܦܝ (BitTorrent Enhancement Proposal)
- **Tracker**: ܛܪܐܟܪ (ܡܫܡܚܢܐ ܕܦܝܪܣ)
- **Swarm**: ܣܘܐܪܡ (ܟܢܘܫܝܐ ܕܦܝܪܣ)

### ܐܣܟܪܝܒܘܬܐ ܕܡܬܦܫܩܢ (Translation)
- **Download**: ܐܚܬܐ / ܢܚܬܐ
- **Upload**: ܣܠܩܐ / ܐܣܩܐ
- **Peer**: ܦܝܪܐ / ܚܒܪܐ
- **Seed**: ܙܪܥܐ (ܦܝܪܐ ܕܫܠܡ ܘܡܦܠܚ)
- **Piece**: ܦܘܪܨܐ / ܓܘܙܐ
- **Chunk**: ܟܘܢܟ (ܦܘܪܨܐ ܕܦܐܝܠܐ)
- **Hash**: ܗܐܫ (ܬܘܕܝܬܐ)
- **Metadata**: ܡܛܕܐܛܐ (ܝܕܥܬܐ ܕܦܐܝܠܐ)
- **Magnet**: ܡܐܓܢܛ (URI ܕܡܬܚܫܒ ܥܡ ܛܘܪܢܛ)

## ܡܠܟܬܐ ܕܪܕܝܢܐ

### Rich Markup Preservation
ܟܠ ܬܓܢܐ ܕܪܝܟ ܡܐܪܟܐܦ ܠܐ ܢܬܦܫܩ:
- `[green]` → `[green]` (ܠܐ ܦܫܩ)
- `[yellow]` → `[yellow]`
- `[bold]` → `[bold]`
- `[cyan]` → `[cyan]`

### Format Placeholders
ܟܠ ܦܠܐܣܗܘܠܕܪܣ ܕܦܘܪܡܐܛ ܠܐ ܢܬܦܫܩ:
- `{name}` → `{name}`
- `{count}` → `{count}`
- `{status}` → `{status}`

### ܐܣܟܪܝܒܘܬܐ ܕܟܘܕܐ
ܟܘܕܐ ܒܐܢܓܠܝܬܐ ܠܐ ܢܬܦܫܩ ܐܠܐ ܢܫܬܒܩ ܒܐܢܓܠܝܬܐ ܘܢܬܦܫܩ ܒܫܪܒܬܐ ܕܐܪܡܝܐ.

### ܡܛܟܣܐ ܕܡܟܕܘܟܣ
- ܡܛܟܣܐ ܕܐܪܡܝܐ ܗܘ ܡܢ ܝܡܝܢܐ ܠܣܡܠܐ (RTL)
- ܐܢܓܠܝܬܐ ܘܐܝܬܝܗ ܡܢ ܣܡܠܐ ܠܝܡܝܢܐ (LTR)
- ܒܡܛܟܣܐ ܕܡܟܕܘܟܣ ܐܝܬ ܕܠܐ ܢܬܚܒܪ ܡܛܟܣܐ
