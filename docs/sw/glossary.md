# Kamusi ya Kiswahili - Swahili Glossary

Hii ni kamusi ya istilahi za BitTorrent na istilahi za kawaida katika ccBitTorrent.

## Istilahi za BitTorrent

### Istilahi za Kukopa (Loanwords)
- **Torrent**: Torrent (tunakubali neno hili kama linavyokuwa)
- **DHT**: DHT (Distributed Hash Table - Jedwali la Hash la Kusambazwa)
- **PEX**: PEX (Peer Exchange - Ubadilishaji wa Peers)
- **BEP**: BEP (BitTorrent Enhancement Proposal - Pendekezo la Uboreshaji la BitTorrent)
- **Tracker**: Tracker (huduma ya kusaidia kupata peers)
- **Swarm**: Swarm (jumuiya ya peers wanaoshiriki torrent moja)

### Istilahi za Kutafsiri
- **Download**: Kupakua / Kupakua
- **Upload**: Kupakia / Kupakia
- **Peer**: Mshirika / Mshirika wa kubadilishana
- **Seed**: Mbegu (peer ambaye amemaliza kupakua na anashiriki)
- **Piece**: Kipande / Kipande
- **Chunk**: Chunk (kipande cha faili)
- **Hash**: Hash (uthibitisho)
- **Metadata**: Metadata (taarifa za faili)
- **Magnet**: Magnet (URI inayohusiana na torrent)

## Miongozo

### Rich Markup Preservation
Lugha zote za Rich markup lazima zibaki kama zilivyo:
- `[green]` → `[green]` (usitafsiri)
- `[yellow]` → `[yellow]`
- `[bold]` → `[bold]`
- `[cyan]` → `[cyan]`

### Format Placeholders
Placeholders zote za muundo lazima zibaki:
- `{name}` → `{name}`
- `{count}` → `{count}`
- `{status}` → `{status}`

### Uandishi
Kiswahili hutumia herufi za Kilatini bila diacritics.

### Uwingi
Kiswahili hutumia viambishi vya uwingi:
- "faili 1" → "faili moja"
- "faili 2" → "faili mbili"
- "faili nyingi" → "faili nyingi"

## Vidokezo

- Tumia lugha ya kawaida (neutral) kwa maeneo ya Afrika Mashariki
- Fuata kanuni za uandishi wa Kiswahili
- Tumia lugha ya amri (imperative) kwa maandishi ya CLI
- Hakikisha tafsiri zako ni sahihi na zinaeleweka
