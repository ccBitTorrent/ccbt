# Awọn Itumọ Yoruba - Yoruba Glossary

Iwe yii ni awọn itumọ ti a yan fun awọn ọrọ BitTorrent ati awọn ọrọ ti o wọpọ ni ccBitTorrent.

## Awọn Ọrọ BitTorrent Pataki

### Awọn Ọrọ Ti a Gba Lati Gẹẹsi (Transliteration)
- **Torrent**: Torrent (a gba ọrọ naa gẹgẹbi)
- **DHT**: DHT (Distributed Hash Table - Tabili Hash Ti a Pinpin)
- **PEX**: PEX (Peer Exchange - Iyipada Peer)
- **BEP**: BEP (BitTorrent Enhancement Proposal - Igbasilẹ BitTorrent)
- **Tracker**: Tracker (ẹrọ ti o ṣe iranlọwọ fun wiwa peer)
- **Swarm**: Swarm (agbegbe awọn peer ti o n ṣe ayipada torrent kan)

### Awọn Ọrọ Ti a Tumọ
- **Download**: Gbigba / Gbigba
- **Upload**: Gbigbe / Gbigbe
- **Peer**: Ẹgbẹ / Ẹgbẹ-ayipada
- **Seed**: Igbasilẹ (peer ti o ti pari gbigba ati ti o n pinpin)
- **Piece**: Apakan / Apakan
- **Chunk**: Chunk (apakan faili)
- **Hash**: Hash (ijẹrisi)
- **Metadata**: Metadata (alaye faili)
- **Magnet**: Magnet (URI ti o jẹmọ torrent)

## Awọn Ilana

### Rich Markup Preservation
Gbogbo awọn Rich markup tags ni a gbọdọ fi silẹ gẹgẹbi:
- `[green]` → `[green]` (ko tumọ)
- `[yellow]` → `[yellow]`
- `[bold]` → `[bold]`
- `[cyan]` → `[cyan]`

### Format Placeholders
Gbogbo awọn format placeholders ni a gbọdọ fi silẹ:
- `{name}` → `{name}`
- `{count}` → `{count}`
- `{status}` → `{status}`

### Diacritics
Yoruba lo awọn diacritics wọnyi:
- à, á, è, é, ì, í, ò, ó, ù, ú
- ẹ, ọ (awọn ọrọ ti o ni awọn lẹta wọnyi)

### Pluralization
Yoruba lo awọn ọrọ ti o yatọ fun ọpọlọpọ:
- "1 faili" → "faili kan"
- "2 faili" → "awọn faili meji"
- "ọpọlọpọ faili" → "ọpọlọpọ awọn faili"

## Awọn Ikilọ

- Lo ọrọ ti o tọ fun ẹkọ (formal) lori ọrọ ti o rọrun (informal)
- Tẹle awọn ilana orthography Yoruba ti o tọ
- Lo awọn diacritics ni gbogbo igba ti o ba tọ
- Ṣe idanwo lati rii daju pe awọn ọrọ ti o tumọ jẹ deede ati ti o ni itumọ
